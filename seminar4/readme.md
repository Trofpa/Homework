4. **Доп. задание №60** Составить частотный словарь элементов двумерного массива  

_Пример:
Если набор данных - таблица (по сути и есть двумерный массив)_
```

4, 6, 1
2, 1, 6
```

```
1 встречается 3 раза
2 встречается 2 раз
3 встречается 1 раз
4 встречается 1 раз
6 встречается 2 раза
```
